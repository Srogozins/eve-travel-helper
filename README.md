eve-travel-helper
=================
EVE Online travel planning helper tool

### Dependencies
* Shell
    * Bash
    * wget
    * unzip
* python
    * sqlite3
    * Testing
        * unittest (testing)
        * mock (testing)
* DB
    * sqlite3

### Planned Features

* Engine
    * List systems
    * Filter systems by field
    * Calculate path(s) with least amount of jumps between two systems
       * Avoid systems with security status above/below threshold
